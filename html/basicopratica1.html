Escrevendo um paragrafo em HTML

Esta Tag define um paragrafo para um texto escrito em uma página
HTML podendo este conter apenas uma ou mais linhas. Perceba tambem
que utilizamos a Tag "br" antes do paragrafo, para oular linha do trecho
anterior para este e tambem na produção do texto para pular as linhas.
Há, um outro detalhe, veja que utilizamos a tag "strong" para grifar em negrito.
